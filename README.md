# ReqLens – AI Requirement Risk Analyzer

ReqLens is an AI-powered tool that analyzes requirement documents to detect:

- Explicit requirements
- Hidden assumptions
- Missing information
- Risk areas
- Clarifying questions
- Overall project risk level

Built using:
- Python
- Streamlit
- Ollama (local LLM)
- ReportLab (PDF export)

This is a proof-of-concept demonstrating AI-assisted delivery intelligence.
